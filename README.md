# Python System Monitor
System monitor GUI written in Python.

![sysmon_screenshot](https://user-images.githubusercontent.com/42113905/60553926-826aa280-9d03-11e9-8602-053e80abe4bc.PNG)

<b>Currently includes:</b>
- CPU %
- Virtual memory %
- Uptime
- System clock
- Boot time
